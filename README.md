# CPCL、TSPL设计
## 简介：
这个项目是一个web应用项目
## 目的：
在网页中使用可拖拽式的元素设计，实现CPCL、TSPL语言，简化热敏打印机的开发流程
## 基础架构：
使用node.js + vue实现
## 功能：
1.页面上使用可拖拽元素，可视化设计标签打印的布局，转化为CPCL、TSPL语言，转化的CPCL、TSPL命令使用全称，如“文本“对应的命令使用”TEXT“
2.需要可定义纸张的宽度、高度
3.设计元素需要包含：文本、直线、条形码
4.纸张的宽度、高度以及元素的坐标，单位均为毫米，程序转化为CPCL语言时，需要转化为热敏打印机对应的单位
5.元素的属性需要可编辑：
    * 文本：字体编号、字体大小、放大倍数
    * 直线：线条粗细
    * 条形码：是否显示条码内容，条码类型（CODE 39、CODE 128），窄条单位宽度、宽条与窄条比例、条码单位高度
6.应用需要有“保存”、“导入”功能，保存、导入文件均为json格式文件
